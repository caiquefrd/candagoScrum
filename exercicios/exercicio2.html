Qual das seguintes afirmações descreve melhor uma Sprint no Scrum?

  • Uma Sprint é um encontro informal onde a equipe discute problemas pessoais.

  • Uma Sprint é um período de tempo fixo e curto, geralmente de 1 a 4 semanas, durante o qual a equipe desenvolve um incremento de trabalho pronto para ser entregue.

  • Uma Sprint é uma reunião de planejamento de longo prazo para definir metas anuais para a equipe de desenvolvimento.

  • Uma Sprint é uma revisão formal de todo o projeto, conduzida mensalmente, para avaliar o progresso geral.
